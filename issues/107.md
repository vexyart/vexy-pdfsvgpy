
## Extra backend

Make Python package extras for Cairo, Poppler, or Ghostscript, so I can install: 

- `vexy-pdfsvgpy[cairo]` allows for hint `ca`, would include https://github.com/Kozea/CairoSVG interpreting SVG and Poppler for interpreting PDF and then creating a Cairo scene, and then serializing that scene into SVG, PDF or bitmaps as needed. 
- `vexy-pdfsvgpy[ghostscript]` allows for hint `gs`, would allow Ghostscript to interpret PDF and then create bitmaps or, more importantly, PDFs, which should include the ability to convert text to paths as needed. 
- `vexy-pdfsvgpy[all]` allows for hints `ca`, `gs`

These won’t be self-contained but instead require additional dependencies. They would never be default, but I could ask for them via hints.
